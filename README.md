# TRACKALYZER

identify visited points of interest in GPX tracks

## usage

### install

```
python3 -m pip install trackalyzer
```

### run

```
trackalyzer <gpx>
```

## development

### prepare

```
python3 -m pip install poetry
```

### build

```
git clone https://github.com/khimaros/trackalyzer

cd trackalyzer

poetry install
```

### run

```
poetry run trackalyzer <gpx>
```
